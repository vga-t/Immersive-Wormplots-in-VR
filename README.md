# Immersive Wormplots in VR

## Overview
Immersive Wormplots in VR is a data visualization project that renders box plots in a virtual reality scene using Babylon.js, Danfo.js, and math.js. The project supports multiple datasets and interactive controls for toggling visibility, scaling, and offset adjustments.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/your-username/Immersive-Wormplots-in-VR.git
   ```
2. Open the project folder:
   ```
   cd Immersive-Wormplots-in-VR
   ```

## How to Run the Project
1. Open the `index.html` file in your browser. The project uses ES modules, so a local development server is recommended.
   - You can quickly start a server with VS Code's Live Server extension or by running:
     ```
     npx http-server .
     ```
2. Once the server is running, navigate to the provided URL in your browser.
