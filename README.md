# Immersive Wormplots in VR

## Overview
Immersive Wormplots in VR is a data visualization project that visualizes multivariate time-series data that contains multiple groups using wormplots in Virtual reality. The project enables wormplot manipulation with interactive VR features for immersive analytics.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/vip-ml/Immersive-Wormplots-in-VR.git
   ```
2. Open the project folder:
   ```
   cd Immersive-Wormplots-in-VR
   ```

## How to Run the Project
1. Open the `index.html` file in your browser.
   - You can start a server with VS Code's Live Server extension or by running:
     ```
     npx http-server .
     ```
2. Once the server is running, navigate to the provided URL in your browser. Enjoy exploring wormplots in VR.
